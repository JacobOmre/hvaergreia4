Hva er greia med redaksjon
NRK Nordland · Hans-Jacob Omre
API-nøkkelen lagres kun i nettleseren din (localStorage) og sendes aldri noe annet sted enn direkte til Anthropic.

Hent nøkkel på console.anthropic.com → API Keys → Create Key.

Du trenger en konto med betalingsinformasjon. Kostnaden er svært lav – typisk noen kroner per dag med normal bruk.
Nøkkelen ser ikke riktig ut. Den skal starte med «sk-ant-».
